📖 Como configurar o Gmail (mais fácil)

  1. Google Account → Segurança → Verificação em 2 etapas (ativar)
  2. Pesquise "Senhas de app" → Criar para "Outro app" → nome: EJ Site
  3. Copie a senha de 16 dígitos e cole no campo Senha acima
  4. Servidor: smtp.gmail.com / Porta: 587 / TLS
