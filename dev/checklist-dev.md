# Checklist de Desenvolvimento — Site da EJ

> Marcar conforme for executando.  
> Última atualização: 01/07/2026

---

## Fase 1 — Pré-desenvolvimento

- [ ] Definir nome da empresa
- [ ] Comprar domínio
- [ ] Criar e-mail profissional (contato@empresa.com.br)
- [ ] Escolher e fechar paleta de cores
- [ ] Criar logo (Figma ou contratar)
- [ ] Levantar todos os textos das páginas (→ ver `/conteudo-paginas/`)
- [ ] Separar fotos dos sócios (fundo neutro, boa qualidade)
- [ ] Levantar prints dos projetos para portfólio

---

## Fase 2 — Setup WordPress

- [ ] Instalar WordPress na hospedagem
- [ ] Criar banco de dados MySQL no cPanel
- [ ] Configurar `wp-config.php`
- [ ] Instalar tema (Astra ou Kadence)
- [ ] Instalar todos os plugins da lista
- [ ] Configurar URLs amigáveis (Configurações > Links Permanentes → Nome do post)
- [ ] Ativar HTTPS / SSL (cPanel → Let's Encrypt)
- [ ] Configurar e-mail SMTP (usar Brevo ou Resend para formulários)
- [ ] Configurar backup automático (UpdraftPlus → Google Drive)

---

## Fase 3 — Criação das páginas

- [ ] Home (template completo com todas as seções)
- [ ] Quem Somos
- [ ] Soluções — Página hub com cards linkando para cada solução
- [ ] Soluções — Desenvolvimento de Sistemas (subpágina)
- [ ] Soluções — Criação de Sites (subpágina)
- [ ] Soluções — Aplicativos Mobile (subpágina)
- [ ] Soluções — Integrações e APIs (subpágina)
- [ ] Soluções — Automações (subpágina)
- [ ] Soluções — Chatbots e IA (subpágina)
- [ ] Soluções — Consultoria de TI (subpágina)
- [ ] Portfólio (criar custom post type "Cases")
- [ ] Blog (configurar categoria, layout de posts)
- [ ] Contato (formulário + WhatsApp + e-mail + mapa se tiver endereço)
- [ ] Trabalhe Conosco

---

## Fase 4 — SEO e Analytics

- [ ] Configurar Yoast SEO
- [ ] Preencher título + meta description de cada página
- [ ] Gerar sitemap XML
- [ ] Criar e verificar conta no Google Search Console
- [ ] Instalar Google Analytics 4
- [ ] Criar e configurar Google Meu Negócio (se aplicável)
- [ ] Configurar Open Graph para redes sociais

---

## Fase 5 — Performance e Qualidade

- [ ] Comprimir todas as imagens
- [ ] Ativar cache (LiteSpeed Cache ou WP Super Cache)
- [ ] Testar no Google PageSpeed Insights (meta: 90+ mobile)
- [ ] Testar em dispositivos móveis reais
- [ ] Testar formulário de contato (envio + recebimento)
- [ ] Testar botão de WhatsApp
- [ ] Testar todos os links internos
- [ ] Revisar todos os textos (erros de português, links faltando)

---

## Fase 6 — Lançamento

- [ ] Fazer backup completo antes de lançar
- [ ] Remover modo manutenção / página "em breve"
- [ ] Anunciar no LinkedIn (Emerson + Julio)
- [ ] Enviar link para rede de contatos via WhatsApp
- [ ] Monitorar Google Search Console na primeira semana

---

## Pós-lançamento (contínuo)

- [ ] Publicar primeiro artigo no blog
- [ ] Adicionar primeiro case de portfólio
- [ ] Pedir depoimento ao primeiro cliente e publicar
- [ ] Publicar no LinkedIn com frequência (1–2x por semana)
- [ ] Monitorar posições no Google mensalmente
