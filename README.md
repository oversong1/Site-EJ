# Site de Vendas — Projeto EJ

> Planejamento completo do site institucional e de vendas da empresa.  
> Última atualização: 01/07/2026

---

## Objetivo

Criar um site profissional que:
1. **Apresenta** quem somos e o que fazemos
2. **Vende** todos os nossos serviços (sistemas, sites, e-commerce, automações, SaaS, ERP, apps)
3. **Gera leads** qualificados via formulário, WhatsApp e CTAs estratégicos
4. **Demonstra autoridade** com blog técnico, cases e portfólio
5. **Converte** visitantes em clientes com copy e estrutura orientados a resultado

---
